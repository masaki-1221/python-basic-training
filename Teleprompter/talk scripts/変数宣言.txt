それでは、変数の宣言についてお話しします。
Pythonでの変数の宣言はとてもシンプルです。
変数名を書いて、値を代入するだけです。
例えば、num という変数に整数の 1 を入れます。
（操作：num = 1 と入力）
次に、文字も入れてみましょう。
name という変数に Mike という文字を入れます。
（操作：name = 'Mike' と入力）
これを出力してみましょう。
（操作：print(num), print(name) を入力して実行）
下に 1 と Mike が表示されますね。
Pythonの特徴として、
いちいち型を宣言しなくてもいいんです。
値を入れた時点で、
Pythonが勝手に型を認識してくれます。
実際にどう認識しているか、
type という関数を使って見てみましょう。
（操作：print(type(num)) 等を入力して実行）
右の方を見てください。
int、つまり整数と表示されています。
Mikeの方は str、文字列ですね。
もし型が分からなくなったら、
この type関数 を使えば正体が分かります。
次は、真偽値、ブーリアンです。
YESかNOか、を表すものです。
（操作：is_ok = True を入力）
先頭は大文字にするのがポイントです。
次に、Pythonの面白い特徴です。
今、整数が入っている num に、
文字の Mike を入れるとどうなるでしょうか。
（操作：num = name を入力して実行）
型が int から str に変わりました。
Pythonでは、箱に入れる中身を変えれば、
箱のタイプも自動的に変わるんです。
次は、型変換です。
Excelなどのデータを扱う時によく使います。
例えば、数字の1が文字として入っている場合。
（操作：name = '1' と入力）
int関数を使うと、無理やり整数に変換できます。
（操作：new_num = int(name) と入力して実行）
ちゃんと int型 になっていますね。
最後に、名前のルールです。
変数名の先頭に数字は使えません。
（操作：1num = 1 と入力してエラーを見せる）
このようにエラーになります。
あと、if などの予約語も使えません。
変数名の付け方には気をつけましょう。
