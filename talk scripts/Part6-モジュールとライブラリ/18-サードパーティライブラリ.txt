次は「サードパーティライブラリ」について説明します。

--------------------
■ サードパーティライブラリとは
--------------------

サードパーティライブラリとは、
Pythonに最初から入っていない、外部で開発されたライブラリです。
pip installコマンドでインストールして使います。

--------------------
■ pip install
--------------------

ライブラリをインストールするには、
コマンドプロンプトで pip install を実行します。

例えば、pandasをインストールする場合は、
pip install pandas と入力します。

生成AIが作るコードに見慣れないimportがあった場合、
pip installでインストールが必要かもしれません。
エラーが出たら、まずライブラリのインストールを確認してください。

--------------------
■ よく使うサードパーティライブラリ
--------------------

業務自動化でよく使われるライブラリを紹介します。

まず、pandasです。
表形式のデータを扱うためのライブラリです。
Excelのデータを読み込んだり、データの集計や加工ができます。

（操作：セルを実行）

次に、openpyxlです。
Excelファイルの読み書きができます。
セルの値を直接操作したり、新しいシートを作成したりできます。

（操作：セルを実行）

次に、requestsです。
Webサイトにアクセスしてデータを取得できます。
API連携でよく使われます。

（操作：セルを実行）

次に、seleniumまたはplaywrightです。
Webブラウザを自動で操作できます。
Webサイトへのログインや、データの入力自動化に使われます。

（操作：セルを実行）

--------------------
■ ライブラリの調べ方
--------------------

ライブラリの使い方が分からない場合は、
公式ドキュメントを確認するか、生成AIに質問するのが効率的です。

また、生成AIにコードを書いてもらう際に、
使用するライブラリを指定することもできます。

--------------------
■ 仮想環境について
--------------------

プロジェクトごとに使うライブラリを分けたい場合は、
仮想環境を使います。

仮想環境を使うと、プロジェクトごとに独立した
ライブラリのインストール先を作ることができます。

詳しい使い方は、必要になった際に調べてみてください。

以上が「サードパーティライブラリ」の説明でした。
