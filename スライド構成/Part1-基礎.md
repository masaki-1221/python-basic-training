## 01. printとコメント

---

### print関数とは

- 文字を画面に出力する関数
- Pythonの最も基本的な機能

```python
print('Hello World')
```

---

### 文字列のルール

- シングルクォーテーション `'` またはダブルクォーテーション `"` で囲む
- どちらを使ってもOK
- **囲まないとエラーになる**

```python
print('Hello')  # OK
print("Hello")  # OK
print(Hello)    # エラー！
```

---

### コメント

- 行の先頭に `#` を付けると、その行は実行されない
- 用途：
  - コードの説明を書く
  - 一時的に処理を無効にする

```python
# これはコメント（実行されない）
print('Hello')  # この部分もコメント
```

## 02. 数値と四則演算

---

### 数値の特徴

- クォーテーションで囲む必要がない
- そのまま計算に使える

```python
print(3)      # 数値の3
print('3')    # 文字列の3（違うもの！）
```

---

### 四則演算

| 演算 | 記号 | 例 |
|------|------|-----|
| 足し算 | `+` | `3 + 7` → 10 |
| 引き算 | `-` | `7 - 3` → 4 |
| 掛け算 | `*` | `3 * 7` → 21 |
| 割り算 | `/` | `6 / 2` → 3.0 |

---

### その他の演算

| 演算 | 記号 | 例 |
|------|------|-----|
| 余り | `%` | `7 % 3` → 1 |
| 切り捨て除算 | `//` | `7 // 3` → 2 |

---

### 注意点

- **全角文字はNG**（プログラム部分は半角で書く）
- クォーテーションで囲むと文字列になる

```python
print(3 + 7)    # 計算される → 10
print('3 + 7')  # 文字列 → "3 + 7"
```

## 03. 文字列

---

### 文字列の結合

- `+` で文字列同士を結合できる
- `*` で文字列を繰り返せる

```python
'Hello' + 'World'  # HelloWorld
'Python' * 3       # PythonPythonPython
```

---

### インデックス

- 文字列から特定の位置の文字を取り出す
- **位置は0から数える**（重要！）

```python
text = 'Python'
text[0]   # P
text[1]   # y
text[-1]  # n（後ろから1番目）
```

| P | y | t | h | o | n |
|---|---|---|---|---|---|
| 0 | 1 | 2 | 3 | 4 | 5 |

---

### スライス

- 文字列の一部分を切り出す
- `[開始:終了]` の形式で指定
- **終了位置の文字は含まれない**

```python
text = 'Python'
text[0:3]   # Pyt
text[:3]    # Pyt（最初から）
text[3:]    # hon（最後まで）
```

---

### 文字列メソッド

| メソッド | 機能 | 例 |
|----------|------|-----|
| `.upper()` | 大文字に変換 | `'hello'.upper()` → `'HELLO'` |
| `.lower()` | 小文字に変換 | `'HELLO'.lower()` → `'hello'` |
| `.strip()` | 前後の空白を削除 | `'  hi  '.strip()` → `'hi'` |
| `.split()` | 分割してリストに | `'a,b,c'.split(',')` → `['a','b','c']` |
| `.replace()` | 置換 | `'Hello'.replace('H','J')` → `'Jello'` |

---

### f-strings（推奨）

- 文字列の中に変数を埋め込む
- 文字列の前に `f` を付ける

```python
name = '山田'
age = 25
print(f'私は{name}、{age}歳です')
# → 私は山田、25歳です
```

## 04. 変数と型

---

### 変数とは

- データを入れておく「箱」
- 箱に名前を付けて、中にデータを保存

```python
name = 'Mike'    # nameという箱にMikeを入れる
number = 5       # numberという箱に5を入れる
```

---

### 変数と文字列の違い

```python
name = 'Mike'
print(name)    # Mike（変数の中身）
print('name')  # name（文字列そのまま）
```

**クォーテーションの有無で意味が変わる！**

---

### 変数の更新

- 後から値を上書きできる
- 自分自身を使った更新も可能

```python
x = 5
x = x + 3   # xは8になる
x += 3      # 省略形（同じ意味）
```

---

### 変数を使うメリット

1. **読みやすさ（可読性）**
   - 変数名で何のデータか分かる

2. **再利用しやすさ**
   - 1箇所変えれば全体に反映

```python
banana_price = 100
banana_count = 3
total = banana_price * banana_count
```

---

### 変数名のルール

| 分類 | 例 | 備考 |
|------|-----|------|
| ✅ 良い | `user_name`, `total_price` | 英単語、アンダースコア区切り |
| ⚠️ 非推奨 | `namae`, `名前` | ローマ字、日本語 |
| ❌ エラー | `1name`, `if` | 数字始まり、予約語 |

---

### データ型

| 型 | 名前 | 例 |
|----|------|-----|
| `int` | 整数型 | `1`, `100`, `-5` |
| `float` | 浮動小数点型 | `3.14`, `0.5` |
| `str` | 文字列型 | `'Hello'`, `"Python"` |
| `bool` | ブール型 | `True`, `False` |

```python
type(100)     # <class 'int'>
type('Hello') # <class 'str'>
```

---

### 型による動作の違い

```python
5 + 7       # 12（計算）
'5' + '7'   # '57'（結合）
```

**同じ `+` でも型によって動作が異なる！**

---

### 型変換

- `str()` → 文字列に変換
- `int()` → 整数に変換
- `float()` → 小数に変換

```python
price = 100
print('価格は' + str(price) + '円')

# f-stringsなら自動変換
print(f'価格は{price}円')
```
